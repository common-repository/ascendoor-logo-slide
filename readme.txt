== Description ==

***Ascendoor Logo Slide*** is a simple logo slide plugin (not a slider) for your clients or partners or any logos showcase. Display responsive logo slide showcase on your/client website.

Ascendoor Logo Slide is a simple logo slide plugin build on top of WordPress REACT backend. So you can live preview your logo slide showcase while editing the slide before you publish.

Ascendoor Logo Slide is easy to setup and configure, responsive, user-friendly, flexible and highly customizable that provides you the ability to customize the style and design with live previewing, no coding skills required.

Ascendoor Logo Slide provides you the shortcode with which you can showcase your logo slide showcase anywhere you want on your site.

== Shortcode ==

<code>[ascendoor_logo_slide]</code>

= With PHP Code =

<code>&lt;?php echo do_shortcode( '[ascendoor_logo_slide]' ); ?&gt;</code>
